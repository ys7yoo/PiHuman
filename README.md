# PiHuman

Let's test and compare human detections of OpenCV and Dlib

# OpenCV version

Nice code from : http://www.pyimagesearch.com/2015/11/09/pedestrian-detection-opencv/

0. Install OpenCV

1. Install required packages 
  ```bash
  pip install imutils
  ```

2. run like this.
  ```bash
  python detectHuman.py -i [IMAGES]
  ```
